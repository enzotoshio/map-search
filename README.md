This project enables you to search addresses by CEP and it uses:

- react
- redux
- redux-thunk
- jest
- enzyme

## Pre requirements

You should have **node v8.x** and **yarn** installed in your machine.

## Available Scripts

In the project directory, you can run:

- `yarn start`
- `yarn test`
- `yarn build`
- `yarn eject`
- `yarn lint`
